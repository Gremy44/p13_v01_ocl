Flake 8
-------

Pour effectuer une vérification, vous pouvez lancer la commande suivante :

.. code-block:: bash

    flake8

Coverage
--------

Pour effectuer un test de couverture, vous pouvez lancer le test avec les commandes suivantes:

   - démarrer le test de couverture

   .. code-block:: bash

      coverage run -m pytest

   - afficher le rapport de couverture

   .. code-block:: bash

      coverage report

   - générer un rapport de couverture au format HTML

   .. code-block:: bash

    coverage html

Pytest
------

Pour les tests unitaire, vous pouvez lancer la commande suivante :

.. code-block:: bash

    pytest
