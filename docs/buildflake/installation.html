3.2. Pré-requis

Pour démarrer le projet, les prérequis suivants sont nécessaires :

  • Un compte GitHub : Lien Github

  • Un compte Docker : Lien Docker

  • Un compte Sentry : Lien Sentry

  • Un compte Heroku : Lien Heroku

  • Un compte Circleci : Lien Circleci

3.3. Repository Github

Vous pourrez retrouver le projet sur le lien suivant : - Lien du repo Github

Cloner le projet sur votre machine : dans votre dossier local de travail, ouvrez un terminal de commande et tapez la commande suivante :

git clone https://github.com/Gremy44/p13_v01_ocl

3.4. Docker

Une fois le projet cloné, installez Docker Desktop sur votre machine et connecter le à votre compte Docker.

Une fois ces étapes faites vous pourrez passer au demarrage de l’application.
