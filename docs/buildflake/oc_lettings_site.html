oc_lettings_site.logger.capture_sentry_message(view_func)[source]

A decorator to capture user visits and exceptions using Sentry.

This decorator wraps a view function and captures a message when the view is visited by a user. If an exception occurs during the view’s execution, the exception is captured and reported to Sentry.

Args:

view_func (function): The original view function to be wrapped.

Returns:

function: The wrapped view function with added Sentry message capturing.

Example:

@capture_sentry_message def my_view(request):

# View logic …
