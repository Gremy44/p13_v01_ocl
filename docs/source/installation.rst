Pré-requis
++++++++++

Pour démarrer le projet, les prérequis suivants sont nécessaires :

- |github-logo| Un compte GitHub : `Lien Github <https://github.com/>`_
- |docker-logo| Un compte Docker : `Lien Docker <https://www.docker.com/>`_
- |sentry-logo| Un compte Sentry : `Lien Sentry <https://sentry.io/>`_
- |heroku-logo| Un compte Heroku : `Lien Heroku <https://www.heroku.com/>`_
- |circleci-logo| Un compte Circleci : `Lien Circleci <https://circleci.com/>`_

Repository Github
+++++++++++++++++

Vous pourrez retrouver le projet sur le lien suivant :
|github-logo| - `Lien du repo Github <https://github.com/Gremy44/p13_v01_ocl>`_

Cloner le projet sur votre machine :
dans votre dossier local de travail, ouvrez un terminal de commande et tapez la commande suivante :

.. code-block:: bash

   git clone https://github.com/Gremy44/p13_v01_ocl

Docker
++++++

Une fois le projet cloné, installez Docker Desktop sur votre machine et connecter le à votre compte Docker.

**Une fois ces étapes faites vous pourrez passer au demarrage de l'application.**
